# Options Volatility Engine

This project implements volatility modeling tools for financial time series and option pricing.

## Features
- Volatility estimation from market data
- Financial time series analysis
- Option volatility modeling

## Technologies
- Python
- NumPy
- Pandas
- Jupyter Notebook

## Files
VolatilityEngine.ipynb – volatility modeling engine  
Volatility.ipynb – financial time series analysis
